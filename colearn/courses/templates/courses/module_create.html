Module Creation Tips

  • Title: Choose a concise and descriptive title that reflects the module's content.
  • Content: Provide clear and detailed content for the module. Use headings, bullet points, and images to improve readability.
  • Video URL: If applicable, include a link to a relevant video to enhance the learning experience.
  • Order: Set the order in which this module should appear within the course to maintain a logical flow.
